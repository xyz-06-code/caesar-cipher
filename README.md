# caesar-cipher
A basic implementation of the Caesar Cipher , one of the oldest encryption methods, where each letter in a message is shifted by a fixed number of postions.

this project has two parts:
- Encrypting a message using a key(1-25)
- Decrypting a message by trying every possible shift (brute force), which shows why this cipher is easy to break

# why i made this
I wanted to actaully understand how basic encryption works. Building the brute force part was the interesting bit, seeing how a message can be cracked just by trying all 25 shifts made it clear why modern encryption needs to be a lot more complex than this.
